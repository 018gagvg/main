Interface Client<O>

User-Facing Sentry SDK Client Client.

This interface contains all methods to interface with the SDK once it has been installed. It allows to send events to Sentry, record breadcrumbs and set a context included in every event. Since the SDK mutates its environment, there will only be one instance during runtime. To retrieve that instance, use {@link Client.getInstance}.

Note that the call to Client.install should occur as early as possible so that even errors during startup can be recorded reliably:

example

import { captureMessage } from '@sentry/node'; captureMessage('Custom message');

    • Defined in core/src/interfaces.ts:202

    Records a new breadcrumb which will be attached to future events.

    Breadcrumbs will be added to subsequent events to provide more context on user's actions prior to an error or crash. To configure the maximum number of breadcrumbs, use Options.maxBreadcrumbs.

    Parameters

    • breadcrumb: Breadcrumb

      The breadcrumb to record.

    • Optional hint: SentryBreadcrumbHint

      May contain additional information about the original breadcrumb.

    • Optional scope: Scope

      An optional scope to store this breadcrumb in.

    Returns void
